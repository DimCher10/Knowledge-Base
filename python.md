# Типы данных
* `int` - integer - целое число
* `float` - число с плавающей точкой
* `bool` - boolean - Булев тип
* `None` - специальный тип, используется при отсутствии значения
* `str` - string -строка
* `list` - список
* `tuple` - кортеж
* `range` - диапазон
* `dict` - `dictionary` ` - словарь
* `set` - множество


# Типы алгоритмов
* линейные
* разветвлённые
    * неполное ветвление (`if...`)
    * полное (`if... else...`)
    * множественное(`if..elif..else..`)
* циклические
    * `for`
    * `while`

# Операции
* `+` - сложение (для чисел)
* `+` - конкатенация (для строк, чисел и т.п)
* `*` - умножение для чисел
* `*` -  повторение для строк
* `%` - остаток от деления
* `//` - целая часть от деления
